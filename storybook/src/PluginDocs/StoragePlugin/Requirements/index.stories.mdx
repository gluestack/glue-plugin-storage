# Requirements

##

<br />

* **Docker** Must be installed on the system.

<br/>

* **Gluestack** must be installed and initialiazed into the project.

<br />

- **Postgres** Plugin Instance
  ```bash
  $ node glue add postgres postgres
  ```

<br />

- **Graphql** Plugin Instance
  ```bash
  $ node glue add graphql graphql
  ```

<br />

- **Minio** Plugin Instance
  ```bash
  $ node glue add minio minio
  ```
