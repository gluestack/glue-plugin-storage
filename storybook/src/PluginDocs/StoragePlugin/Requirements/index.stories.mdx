# Requirements

##

**Docker** must be installed on the system.

**Postgres** Plugin Instance, you can add one using the following command:
```sh
$ node glue add postgres postgres
```

**Graphql** Plugin Instance, you can add one using the following command:
```sh
$ node glue add graphql graphql
```

**Minio** Plugin Instance, you can add one using the following command:
```sh
$ node glue add minio minio
```
